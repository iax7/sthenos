# Icon Components

Reusable icon components extracted from inline SVGs.

Available:

* `IconPlus`
* `IconEdit`
* `IconTrash`
* `IconMenu`
* `IconDownload`
* `IconUpload`
* `IconClear`

Usage example:

```vue
<script setup>
import IconPlus from '@/components/icons/IconPlus.vue'
</script>
<template>
  <button class="icon-btn">
    <IconPlus />
    <span>New Entry</span>
  </button>
</template>
```
